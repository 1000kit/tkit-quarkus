# tkit-quarkus-context

Context propagation helpers used by other extensions like logging or jpa.

Maven dependency
```xml
<dependency>
    <groupId>org.tkit.quarkus.lib</groupId>
    <artifactId>tkit-quarkus-context</artifactId>
</dependency>
```

Context object holds

* request ID
* principal name
* principal token
* business values
