# tkit-quarkus-log

1000kit Quarkus logger extension, core log library for Quarkus ARC `cdi` implementation.

Maven dependency
```xml
<dependency>
    <groupId>org.tkit.quarkus.lib</groupId>
    <artifactId>tkit-quarkus-log-cdi</artifactId>
</dependency>
```
